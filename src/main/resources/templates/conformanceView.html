GeoTime WFS Service Conformance

Landingpage > GeoTime WFS Service landingpage > Conformance

The GeoTime WFS Service is conformed to:

  • Core of OGC API Common 1.0
  • Collections of OGC API Common 1.0
  • OAS3 of OGC API Common 1.0
  • Core of OGC API Records 1.0
  • OAS30 of OGC API Records 1.0
  • HTML of OGC API Records 1.0
  • JSON of OGC API Records 1.0
